# OTP Security Service - Сервис защиты операций с временными кодами

## Описание проекта

Сервис обеспечивает дополнительный уровень безопасности при выполнении различных операций с помощью одноразовых временных кодов (OTP). Позволяет генерировать, отправлять и проверять коды подтверждения через различные каналы связи.

## Технологический стек

- **Backend**: Java 11+, com.sun.net.httpserver
- **База данных**: PostgreSQL 17, JDBC
- **Аутентификация**: JWT токены
- **Каналы доставки**: Email (SMTP), SMS (эмулятор SMPP), Telegram Bot, файл
- **Сборка**: Gradle
- **Фронтенд**: HTML/CSS/JavaScript
